# 跨场景少样本无线信道建模项目主页

这是 GitHub Pages 可直接发布的中文学术项目主页。

## 文件结构

```text
index.html
assets/style.css
assets/script.js
.nojekyll
README.md
```

## 发布方法

1. 把本目录所有文件复制到你的 GitHub Pages 仓库根目录。
2. GitHub 仓库进入 Settings → Pages。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，Folder 选择 `/root`。
5. 保存后等待 Pages 自动发布。

## 发布前建议替换

- `index.html` 中的 GitHub 链接：默认写成 `https://github.com/accuseoneee/wireless_pretrain`。
- `PDF` 链接：目前是占位 `#`，上传论文 PDF 后替换。
- 头像：当前是 SVG initials，可替换为你的正式照片。
- 如果未来论文投稿/录用，可以把 `Manuscript in preparation` 改成具体 venue。

## 设计定位

页面参考学术个人主页的信息组织方式：左侧身份栏，右侧 About、Research Question、Method、Selected Results、Evidence Map、Paper & Code。默认低信息密度，细节通过按钮、标签页、弹窗展开。
